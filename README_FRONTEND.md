# AI虚拟形象聊天系统 - 前端应用

这是一个基于Flask和WebRTC的现代化AI虚拟形象聊天系统前端应用。

## 功能特性

- 🎥 **实时视频通话**: 支持摄像头和麦克风
- 🤖 **AI虚拟形象**: 多种虚拟形象类型支持
- 🎤 **语音识别**: 实时语音转文字
- 💬 **文字聊天**: 支持文字输入对话
- 🔧 **设备管理**: 音频/视频设备切换
- 📱 **响应式设计**: 支持桌面和移动设备
- 🌙 **深色模式**: 自动适应系统主题
- ⚡ **实时通信**: WebSocket实时数据交换

## 文件结构

```
├── app.py                 # Flask主应用
├── run_app.py            # 启动脚本
├── requirements.txt      # Python依赖项
├── templates/
│   └── index.html       # 主页面模板
└── static/
    ├── css/
    │   └── style.css    # 样式文件
    └── js/
        └── app.js       # JavaScript应用逻辑
```

## 安装和运行

### 1. 安装依赖项

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
python run_app.py
```

或者直接运行：

```bash
python app.py
```

### 3. 访问应用

打开浏览器访问：`http://127.0.0.1:8282`

## 配置说明

应用使用 `config/glut2.yaml` 配置文件，包含以下主要配置：

- **服务配置**: 主机地址和端口
- **AI模型配置**: 大语言模型设置
- **虚拟形象配置**: 虚拟形象类型和参数
- **音频/视频配置**: 媒体设备设置

## 使用说明

### 基本操作

1. **开始对话**: 点击"开始对话"按钮启动摄像头和麦克风
2. **语音对话**: 按住语音按钮说话，或使用空格键
3. **文字聊天**: 在输入框中输入文字消息
4. **设备控制**: 使用底部控制按钮开关麦克风和摄像头
5. **设置**: 点击右上角设置按钮配置设备和模型

### 快捷键

- `Ctrl + Enter`: 发送文字消息
- `Space`: 按住说话
- `Esc`: 关闭模态框

### 设备管理

- **音频设备**: 在设置中选择麦克风设备
- **视频设备**: 在设置中选择摄像头设备
- **虚拟形象**: 选择虚拟形象类型（LiteAvatar、LAM、MuseTalk）
- **AI模型**: 选择大语言模型（百炼、MiniCPM、OpenAI兼容）

## 故障排除

### 常见问题

1. **摄像头无法访问**
   - 确保浏览器允许摄像头权限
   - 检查摄像头是否被其他应用占用

2. **麦克风无法访问**
   - 确保浏览器允许麦克风权限
   - 检查系统音频设置

3. **WebSocket连接失败**
   - 检查防火墙设置
   - 确保端口8282未被占用

4. **页面加载缓慢**
   - 检查网络连接
   - 清除浏览器缓存

启动时添加调试信息：

```bash
python app.py --debug
```

## 开发说明

### 添加新功能

1. **后端API**: 在 `app.py` 中添加新的路由
2. **前端界面**: 在 `templates/index.html` 中添加HTML元素
3. **样式**: 在 `static/css/style.css` 中添加CSS样式
4. **逻辑**: 在 `static/js/app.js` 中添加JavaScript代码

### 自定义虚拟形象

1. 在 `AIProcessor` 类中添加新的虚拟形象处理逻辑
2. 在前端添加相应的动画效果
3. 更新配置文件中的虚拟形象选项
